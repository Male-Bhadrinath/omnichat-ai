## 📌 Problem Statement

Businesses receive customer enquiries across **WhatsApp, Instagram, Facebook, LinkedIn, phone calls, websites**, and other platforms. Managing conversations manually causes:

- ❌ Delayed responses (avg. 12+ hours)
- ❌ Missed leads and lost revenue
- ❌ Inconsistent communication tone
- ❌ Poor customer experience

Companies need a **unified conversational AI system** capable of handling messages, voice calls, collecting data, booking appointments, and answering queries automatically — 24/7.

---

## 💡 Solution

**OmniChat AI** is a unified conversational AI platform that brings all customer communication channels into a single intelligent dashboard. It automates responses, captures leads, books appointments, and provides deep analytics — empowering SMBs with enterprise-grade automation.

## 🖥️ Setup Instructions

### Prerequisites
- Python 3.9+
- pip

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/omnichat-ai.git
cd omnichat-ai
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the App
```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`

### 4. Deploy to Streamlit Cloud (Recommended)
1. Push your code to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repo
4. Set main file to `app.py`
5. Click **Deploy** — your app is live in minutes!
